ISkill Methods
ISkill Interface See Also Send Feedback

The ISkill type exposes the following members.

Methods

  NameDescription
GetCooldown
returns the skill cooldown in seconds for a specified level
GetIconTexture
Return the corresponding icon.
GetName
Return the skill name.
GetNeededEnergy
returns the needed energy for a specified level the skill will not be executed if the energy is lower
GetNeededItems
special items that are needed to learn a certain skill
GetNeededSkills
returns a list of skill names that are needed for this very skill to be learned
IsPassive
Return whether a skill is active (like attacks) or passive (walk speed, jump height, ...)
OnLevelChange
Called when the skill goes to another level.
OnUse
called when used
OnUseWithoutMana
similar to OnUse but only called when the player has not enough mana for the skill when returning true, the skill will be set on cooldown BUT no mana will be drained this can be used to specify f.e. fallback skills (electro fist may just use fist when low on mana)
